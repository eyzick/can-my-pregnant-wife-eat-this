A simple, mobile-friendly web app to help pregnant women quickly check if a specific food is safe to eat. It uses real-time Google Search API analysis to provide "Likely Safe", "Caution", or "Not Safe" recommendations.

## Features

- **Real-time Safety Check:** Analyzes Google Search results for keywords like "avoid", "unsafe", "listeria", etc.
- **Smart Heuristics:** Distinguishes between "Raw" warnings (Unsafe) and "Cooked" instructions (Safe/Caution).
- **Wikipedia Integration:** Provides general food information and images.
- **Mobile First:** Designed for quick checks on the go.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Google Custom Search API
- Wikipedia API
